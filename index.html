Why Linux?

It's open source

Other open source stuff runs on it

It's lightweight

It runs on almost anything

It's free

  • Kernal is GPL v2, anyone can use it and add to it
  • Almost all open source projects (and apache ones) run on it as a first citizen
  • Doesn't have a GUI by default
  • It could even run on a real IoT toaster, arduinos, pis, multi core mega machines
  • It's free as in "free speech," not as in "free beer" :p

History

It started with Unix in the 1970s

Multi User, Multitasking Operating System

Dumb terminals

Built around the "Unix philosophy"

  • Came out of AT&T, Apple Macs run it
  • Concept of multi users and multitasking built in, Daemons
  • Original started out on mainframe systems, powerful server with dumb terminals. Though Sparc machines existed
  • Things should do one job and do it well

Birth of Linux in the 1990s

Linus Torvalds

Runs on x86 (originally 386)

Ported a lot of programs from unix

  • Small side project
  • Still involved in kernel project
  • So you could run it on any PC pretty early on
  • bash and gcc to start of with, then x-windows then loads after that
